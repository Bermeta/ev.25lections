Linux Commands 

ctrl + alt + t = команда открытия терминала

~ - символ домашней директории
/ - символ корневой директории 

.. - символы обозначают предыдующию директорию
. - символ обозначает текущую диреткорию 


1. ls (list) - команда для вывода всех директорий (папок) и файлов в текущей (рабочей) директории в терминала 
2. cd (change directory) - команда для изменения или переходов с одной директории на другую 
 cd <название директории>
 3. mkdir (make directory) - создает указанную  директорию в рабочей директории в терминале
 mkdir <название директории>
4. touch - создает указанный файл 
5. rm (remove) - удаление файлов 
6. rmdir (remove directory) - удаление директории 
7. pwd (present working directory) - выводит пусть до текущей директории
8. rm -rf - принудительное удаление
9. ls -a - выводит список как скрытых так и обычных файлов и директорий
10. nano <path_to_file> - откроется файл для редактирования в графическом интерфейсе
*если нет файла, то  nano попытается создать новый файл
11. cat <file_name> - выводит содержимое файла в терминале 
12. echo - команда для быстрой записи текстовой информации в какой либо файл. 
echo text_для записи > название_файла

 mv (move) - команда для переименовывания файла и директории, а также для перемещения файлов и директории. 

 переименовывание: mv jonh  john 
		mv <старое_название> <новое название файла>
Перемещение: mv john ~/Desktop/
		mv <file/directory> <directory>/
